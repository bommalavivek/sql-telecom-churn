# SQL Telecom Churn Analysis

This project demonstrates SQL data analysis using PostgreSQL on a telecom customer churn dataset.

## Tools Used

- PostgreSQL
- pgAdmin 4
- Git & GitHub

## Project Overview
This project analyzes a telecom customer churn dataset using PostgreSQL. The objective is to identify factors affecting customer churn and answer key business questions using SQL.

The project demonstrates SQL skills ranging from basic queries to advanced concepts such as joins, window functions, subqueries, and Common Table Expressions.

## Dataset
Dataset: Telecom Customer Churn
Records: 7,043 customers
Columns: 21
Database: PostgreSQL

The dataset includes:

- Customer demographics
- Contract information
- Internet services
- Billing details
- Payment methods
- Customer churn status
